# Releap Firebase Functions

## Setup

### Install Firebase CLI

```bash
$ npm install -g firebase-tools
```

### Login to Firebase

```bash
$ firebase login
```

### Install dependencies

```bash
$ cd functions && npm install
```

### Deploy (Staging)

```bash
$ cd functions && npm run deploy:staging
```

### Deploy (Production)

```bash
$ cd functions && npm run deploy
```
